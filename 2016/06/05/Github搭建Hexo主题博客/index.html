##Github搭建Hexo主题博客##
hexo出自台湾大学生tommy351之手，是一个基于Node.js的静态博客程序，其编译上百篇文字只需要几秒。hexo生成的静态网页可以直接放到GitHub Pages，BAE，SAE等平台上


对于hexo博客来说，静态html页面使得加载速度变得十分快，同时支持markdown书写，很多主题样式也极其简约，符合程序员的风格。
下面给出几个搭建该套博客的文章，希望对大家搭建有所帮助！
