C++学习笔记

friend（友元）

相同class的各个objects互为friends（友元）
即可以直接调用private的成员变量，方法
数据尽量放在private里

const成员函数

构造函数可以有多个，重载
用初始化列表
double real( ) const { return re; }
不改变数据内容
const 成员函数(const的作用：说明其不会修改数据成员)
任何不会修改数据成员的函数都应该声明为const 类型。如果在编写const 成员函数时，不慎修改了数据成员，或者调用了其它非const 成员函数，编译器将指出错误，这无疑会提高程序的健壮性
关于Const函数的几点规则：
a. const对象只能访问const成员函数,而非const对象可以访问任意的成员函数,包括const成员函数.
b. const对象的成员是不可修改的,然而const对象通过指针维护的对象却是可以修改的.
c. const成员函数不可以修改对象的数据,不管对象是否具有const性质.它在编译时,以是否修改成员数据为依据,进行检查.

参数传递 pass by value OR pass by reference(to const)

引用 &
尽量不要传值！
如果不想改动值使用 const XXX&
返回值的传递也尽量传reference
若在函数里创建局部变量，return不能传引用

任何成员函数都一个this pointer 指向调用者
return by reference 传递者无需知道接收者是以reference形式接收
